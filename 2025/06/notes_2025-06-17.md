#### calculate avereage of absolute_ic50 value to compare to each individual row of the absolute_ic50
```sql
SELECT
    formatted_id,
    assay_type,
    cell_line,
    cells_well,
    time_hr,
    compound_status,
    absolute_ic50,
    AVG(absolute_ic50) OVER (
        PARTITION BY
            formatted_id,
            assay_type,
            cell_line,
            cells_well,
            time_hr,
            compound_status
    ) AS avg_absolute_ic50,
    absolute_ic50 - AVG(absolute_ic50) OVER (
        PARTITION BY
            formatted_id,
            assay_type,
            cell_line,
            cells_well,
            time_hr,
            compound_status
    ) AS deviation
FROM
    ds3_userdata.kat6a_registry_summary
WHERE
    compound_status IS NOT NULL
```
